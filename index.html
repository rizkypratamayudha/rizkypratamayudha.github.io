Backend Developer

Hello! Saya adalah Mahasiswa Politeknik Negeri Malang yang berfokus pada Backend Developer dengan pengalaman dalam membangun aplikasi server-side yang scalable, aman, dan berkinerja tinggi. Saya ahli dalam menggunakan teknologi seperti Laravel, MySql, Javascript, PHP Native, API yang digunakan pada aplikasi Mobile Flutter.

Passion saya adalah mengoptimalkan kinerja aplikasi dengan solusi yang efisien. Saya juga memiliki pengalaman dalam menggunakan teknologi frontend seperti HTML 5, CSS, Bootstrap 5, Tailwind CSS, Templating AdminLTE

Saya percaya bahwa kerja tim dan kolaborasi adalah kunci untuk menciptakan solusi terbaik bagi klien dan pengguna akhir.
